Frutería y verdulería
" La Huerta "
Descripción de información:

La frutería y verdulería “La Huerta” solicita tus servicios para realizar una base de datos para el control de sus clientes, proveedores, productos y facturas, por lo que te plantea sus flujos de información.

  • La frutería desea registrar la información de todos sus clientes. La información que desea guardar de sus clientes es el ID del cliente, su nombre, teléfono y estado civil. También se tiene un registro de las ciudades donde radican sus clientes, por lo que se almacena el ID de la ciudad y su nombre. 
  • La frutería también desea almacenar la información de sus proveedores, como el RFC del proveedor, nombre del proveedor y teléfono; así como de sus productos, como el ID del producto, nombre, clasificación y precio al público. 
  • Para el control de sus inventarios desea almacenar las notas de venta a clientes y las facturas de compra a proveedores. En las notas de venta desea guardar el ID de la nota, la fecha, el ID de los productos vendidos y la cantidad. En las facturas de compra a proveedores se desea almacenar el ID de la factura, la fecha, el ID de los productos comprados, el costo y la cantidad.  
          Descripción de las relaciones:
  • Cada cliente radica en una ciudad, pero en una ciudad pueden radicar varios clientes. 
  • Cada cliente puede tener varias notas de venta, pero cada nota es de un solo cliente.
  • Cada nota de venta puede tener el registro de venta de uno o varios productos, y un mismo producto puede ser vendido en distintas notas. 
  • Cada proveedor puede tener varias facturas de compra, pero cada factura es de un solo proveedor.
  • Cada proveedor puede surtir varios productos y cada producto puede ser surtido por distintos proveedores. 
  • Cada factura de compra puede tener el registro de compra de uno o varios productos, y un mismo producto puede ser comprado en distintas facturas.
