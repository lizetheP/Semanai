Laboratorio 8.  ACCESS Controles 

 Objetivos:
 
           El alumno será capaz de:
     
    • Crear controles simples y avanzados
    • Modificar las propiedades de los controles
    • Crear etiquetas
    • Crear cuadros combinados de alguna tabla donde se muestre una lista con la clave y descripción del campo
    • Crear cuadros combinados de alguna tabla donde se muestre sola la descripción del campo
    • Crear cuadros combinados donde se escriban los valores de la lista
    • Crear grupos de opciones

    Instrucciones:
 
    • Crea dentro de la unidad d:/ de tu computadora una carpeta con tu matrícula.
    • Guarda en esa carpeta la base de datos LaHuertaControl_matricula.accdb con tu matrícula.
    • Realiza la lectura de los capítulos:
      • Capítulo 4. Formularios 
      • Capítulo 5. Controles
.
  CREACIÓN DE CUADROS COMBINADOS DONDE SE MUESTRE UNA LISTA CON LA CLAVE Y DESCRIPCIÓN DEL CAMPO

       1. CUADRO COMBINADO DEL CAMPO ID_CIUDAD DEL FORMULARIO DE CLIENTES 

Utiliza el formulario de Clientes para crear un cuadro combinado que se presente una lista con el ID y nombre de las ciudad en la que radica un cliente.


    .
    • Abre el formulario de Clientes, dando doble clic sobre este.


    • Abre el formulario de Clientes en vista diseño Ver > Vista diseño
    • Selecciona el cuadro de texto de ID_Ciudad y bórralo.
    • Debajo del campo Teléfono, crea un cuadro combinado  para el campo de Ciudad, el cual muestre una lista con el ID y nombre de la ciudad para que el usuario pueda seleccionar automáticamente una opción de la lista.
    • Ordena la información del cuadro combinado por el campo ID_Ciudad ascendentemente.
    • Checa tu cuadro combinado dando clic en Ver > Vista Formulario
  CUADROS COMBINADOS DONDE SE ESCRIBAN LOS VALORES DE LA LISTA

      2. CUADRO COMBINADO DEL CAMPO CLASIFICACIÓN DEL FORMULARIO DE PRODUCTOS

    Utiliza el formulario de Productos para crear un cuadro combinado donde se presenten las distintas clasificaciones de los productos (Frutas, Frutos secos, Granos y cereales y Verduras).
     
    • Abre el formulario de Productos, dando doble clic sobre este.
    • Abre el formulario de Productos en vista diseño Ver > Vista diseño
    • Selecciona el cuadro de texto de Clasificación que se encuentra ubicado dentro de la sección de Detalle y bórralo:

        

    • En la sección de Detalle, crea un cuadro combinado  para el campo de Clasificación, el cual muestre una lista con los posibles valores que pueden ser capturados en el campo de Clasificación, y del cual se pueda seleccionar automáticamente una opción. 
    • Los valores del campo Clasificación deberán ser los siguientes: 
      • Frutas
      • Frutos secos
      • Granos y cereales
      • Verduras
    • El nombre de la etiqueta del cuadro combinado deberá ser Clasificación. .
    • En la sección de Detalle, selecciona la etiqueta de Clasificación que aparece a un costado del cuadro combinado y bórrala.

      .

    • Checa tu cuadro combinado dando clic en Ver > Vista Formulario

      3. CUADRO COMBINADO DEL CAMPO TIPO DE PAGO DEL FORMULARIO DE NOTAS DE VENTA

Utiliza el formulario de Notas de venta para crear un cuadro combinado donde se presenten los distintos tipos de pago (Efectivo, Tarjeta o Vales).

  • Abre el formulario de Notas de venta, dando doble clic sobre este y luego ve a la vista diseño Ver > Vista diseño 
  • Selecciona el cuadro de texto de Tipo de pago y bórralo.

     
  • En la sección de Detalle, crea un cuadro combinado  para el campo de Tipo de pago, el cual muestre una lista con los posibles valores que pueden ser capturados en este campo, y del cual se pueda seleccionar automáticamente una opción. 
  • Los valores del campo Tipo de pago deberán ser: Efectivo, Tarjeta y Vales. 
  • El nombre de la etiqueta del cuadro combinado deberá ser Tipo de pago.
    • .
       
  • Checa tu cuadro combinado dando clic en Ver > Vista Formulario

      4. CUADRO COMBINADO DEL CAMPO RFC_PROVEEDOR

Utiliza el formulario de Facturas de compra para crear un cuadro combinado donde se muestre una lista con solamente el nombre del proveedor.



    • Abre el formulario de Facturas de compra, dando doble clic sobre este.


    • Abre el formulario de Facturas de compra en vista diseño Ver > Vista diseño
    • Selecciona el cuadro de texto de RFC_Proveedor y bórralo
    • Crea un cuadro combinado  para el campo RFC_Proveedor, el cual incluya el RFC del proveedor y su nombre, aunque solamente se muestre el nombre del proveedor, y del cual se pueda seleccionar automáticamente una opción. El cuadro combinado deberá almacenar el campo RFC del proveedor, aunque solo se muestre el nombre del proveedor.
    • Ordena la información del cuadro combinado por el campo Nombre del proveedor ascendentemente.
    • Checa tu cuadro combinado dando clic en Ver > Vista Formulario
    
      5. CUADRO COMBINADO DEL CAMPO ID_CLIENTE

Utiliza el formulario de Notas de venta para crear un cuadro combinado donde se muestre una lista con solamente el nombre del cliente.

  • Abre el formulario de Notas de venta, dando doble clic sobre este y luego ve a la vista diseño Ver > Vista diseño.
  • Selecciona el cuadro de texto de ID_Cliente y bórralo
  • Crea un cuadro combinado  para el campo ID_Cliente, el cual incluya el ID del cliente y su nombre, aunque solamente se muestre el nombre del cliente, y del cual se pueda seleccionar automáticamente una opción.
  • El cuadro combinado deberá almacenar el campo ID del cliente, aunque solo se muestre el nombre del cliente.
  • Ordena la información del cuadro combinado por el campo Nombre del cliente ascendentemente.
  • Checa tu cuadro combinado dando clic en Ver > Vista Formulario
